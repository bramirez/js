## What is an Object?

In everyone's mind:
* An instance of a class

In Javascript:
* A bag of properties (key and value)
* hash-like (ruby analogy)

```js
var myObject = {key1 : 'value',
                key2 : 100,
                key3 : true,
                key4 : function(){ alert('Hello'); }};
```

## 2 Ways To Access Properties

The dot `.` operator.
```js
myObject.key1;
myObject.key4();
```

The array-hash-thingy-whatchamacallit operator `[]`.
```js
myObject['key1'];
myObject['key4']();
```


## A More Useful Object

```js
var john = {first_name : 'John',
            last_name : 'Frusciante',
            age : 43,
            playGuitar : function(){
              alert("Strum Strum Strum");
            }};
            

john.playGuitar();
alert( john.first_name + ' is ' + john.age + ' years old.');
```
