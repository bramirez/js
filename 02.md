## What is an Object?

In everyone's mind:
* An instance of a class

In Javascript:
* A bag of properties (key and value)
* hash-like (ruby analogy)

```js
var myObject = {key1 : 'value',
                key2 : 100,
                key3 : true,
                key4 : function(){ alert('Hello'); }};
```

## 2 Ways To Access Properties

The dot `.` operator.
```js
myObject.key1;
myObject.key4();
```

The array-hash-thingy-whatchamacallit operator `[]`.
```js
myObject['key1'];
myObject['key4']();
```
