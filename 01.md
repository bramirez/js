## Comparison

* `==` - is not so reliable. Doesn't check if 2 expression has the same type.
* `===` - checks if 2 expression has the same value and the same type.

```js
  null == undefined // true
  null === undefined // false
  
  "" == false // true
  "" === false // false
  
  1 == true // true
  1 === true // false
  
  [] == false // true
  [] === false // false
```
